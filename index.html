tinyCode_Fun With Arrays

Object: Use the buttons to unscramble the word in as few moves as possible.

Moves:

How to Play

Pop/Push

Removes the last letter from active box and adds the letter to the end of the inactive box

Shift/Pop

Removes the first letter from the active box and adds the letter to the end of the inactive box

Pop/Unshift

Removes the last letter from the active box and adds the letter to the start of the inactive box

Shift/Unshift

Removes the first letter from the active box and adds the letter to the start of the inactive box

Reverse

Reverse the order of the letters in the active box

Sort

Sorts the letters in the active box, ascending

Splice

Start At

Index position/location where Splice will begin.

Length

Number of letters to cut from active word. Pushes the spliced letters from the active box to the inactive box

A site created to learn more about JavaScript array methods.

by Jeremy Uriz

I recently discovered how much more I learn (and enjoy learning) by creating projects from the ground up. In this case, I wanted to learn more about arrays in JavaScript. During the course of this project I learned (and forgot) more about arrays than I had in the previous 6 months. The bulk of this code was written over 3 days in June of 2020.
